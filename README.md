Uma análise exploratória de dados utilizando Python, Pandas e Plotly para investigar padrões geográficos e temporais do tubarão-raposa-olhudo através de um dataset público do GBIF.

Objetivos
- Explorar a distribuição global da espécie
- Investigar possíveis padrões migratórios
- Identificar tendências temporais
- Analisar possíveis influências climáticas
